# Project Title
點選某位苦主，產生對他說的幹話。

# Installing
1.下載nonsense_generator資料夾

```
$ git clone https://github.com/YayunCheng86/nonsense-generator.git
```

2.開啟Terminal，進入此專案

```
cd trash_talk_generator
```

3.使用npm安裝所有相依套件

```
$ npm install
```

4.執行專案

```
$ npm run dev
```

並開啟網址在本機端
`http://localhost:3000`

# Features
- 使用者可以在工程師、設計師、或創業家中選擇一個對象（三選一)
- 使用者按下【產生幹話】後，textbox 就會出現幹話
